Ensamble de una computadora


Hardware


Ahora que ya quedo claro que es el hardware y software, sabemos que el armar una computadora con lleva todo que tiene que ver con el hardware. Cabe aclarar que al armar un computadora solo es la parte que podemos "tocar", ya que lo que la hace funcionar es el Sistema Operativo, que es software.

Ahora, los componentes basicos para que una computadora pueda funcionar son los siguientes:

★ Tarjeta madre: Encargada de conectar entre si todos los componentes.
★ Procesador (CPU): Encargado de procesar todas las tareas que se le asigne.
★ Memoria RAM: Se encarga de mantener abiertos los procesos.
★ Almacenamiento: Se encarga de almacenar toda información, inclusive el sistema operativo.
★ Fuente de poder (PSU): Encargada de darle la potencia a la computadora.
★ Tarjeta gráfica (GPU): Esta es opcional, es la encargada de dar video a la computadora.
